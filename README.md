# AI-Based Epidemic Prediction and Prevention System

A Python-based epidemic prediction system developed using Artificial Intelligence concepts to simulate disease spread, analyze outbreak severity, and recommend prevention strategies.

## Features

* Epidemic spread simulation using population data
* Outbreak risk analysis and prediction
* Hidden outbreak detection using HMM
* AI-based decision making using MDP
* Prevention strategy recommendations
* Disease spread visualization through graphs
* Support for multiple population scenarios

## AI Concepts Used

* Hidden Markov Models (HMM)
* Markov Decision Processes (MDP)
* Probabilistic Analysis
* Disease Spread Simulation
* Risk Assessment and Prediction

## Technologies Used

* Python
* NumPy
* Matplotlib
* Artificial Intelligence Principles

## How to Run

```bash
pip install numpy matplotlib
python main.py
```
